hexo搭建教程

Hexo简介

Hexo是搭建博客的框架

  1. 安装Git
  2. 安装nodejs
  3. 安装hexo 前面git和nodejs安装好后，就可以安装hexo了，你可以先创建一个文件夹blog(随意命名)，在这个文件夹下直接右键git bash打开 输入命令 hexo s 也就是hexo server 代表打开了本地服务在浏览器输入localhost:4000就可以看到你生成的博客了
  4. GitHub创建个人仓库
    例如用户名是zhangsan 仓库名就是 zhangsan.github.io
